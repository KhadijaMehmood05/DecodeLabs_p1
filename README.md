# Number Guessing Game (Java)

## About the Project
This is a simple console-based Number Guessing Game developed in Java. The program generates a random number between 1 and 100, and the user has to guess it within a limited number of attempts.

The game gives feedback after each guess to help the player reach the correct number.

## How the Game Works
- A random number is generated between 1 and 100
- The player gets 7 attempts to guess the number
- After each guess, the program gives hints:
  - "Too High" if the guess is greater than the number
  - "Too Low" if the guess is less than the number
  - A "very close" hint if the guess is near the correct number
- Score is calculated based on remaining attempts
- The player can choose to play again

## Features
- Random number generation
- Input validation using try-catch
- Range checking (only numbers from 1 to 100 allowed)
- Limited attempts system
- Hint system for better user experience
- Score calculation
- Option to replay the game

## Technologies Used
- Java
- Scanner class for input handling
- Random class for number generation
- Eclipse IDE

## Project Purpose
This project was created to practice basic Java concepts such as loops, conditions, exception handling, and user input.
